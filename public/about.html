Sobre

O Bot-Contador é uma ferramenta desenvolvida para auxiliar administradores de grupos no WhatsApp a monitorar a quantidade de membros em seus grupos. Com ele, você pode automatizar o acompanhamento do número de integrantes, facilitando a gestão e o controle do crescimento do grupo.

Leia mais sobre o projeto no GitHub.

Como começar a usar?

  1. Clone o repositório: Baixe o código-fonte do Bot-Contador do GitHub.
  2. Instale as dependências: Use o comando npm install para instalar todas as dependências necessárias.
  3. Configure o bot: Edite o arquivo config/config.example.json para ajustar os grupos monitorados, o número de alerta e os limites.
  4. Inicie o bot: Dê um duplo-clique no arquivo LIGAR_BOT.bat (no Windows) ou execute o comando npm start no terminal.
  5. Aguarde: O bot irá iniciar e você verá mensagens de log no terminal. Isso leva em média 30s.
  6. Acesse o painel de controle: Abra o navegador e vá para http://localhost:3000 para acessar o painel do bot.
  7. Conecte ao WhatsApp: No painel de controle ou no terminal, escaneie o QR Code exibido para conectar o bot à sua conta do WhatsApp.
  8. Monitore a atividade: Acompanhe os logs e os alertas sobre a contagem de membros diretamente no painel.
